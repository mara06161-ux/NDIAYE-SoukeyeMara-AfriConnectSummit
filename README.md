## Description du projet

Site vitrine complet pour "AfriConnect Summit", une conférence tech panafricaine fictive réunissant développeurs, entrepreneurs et investisseurs du continent. Le site comprend 4 pages : Accueil, Programme, Intervenants et Contact, conçu dans un style moderne avec dark mode, animations et interactivité complète.

## Technologies utilisées

- **HTML5** — structure sémantique, accessibilité (alt, aria-label, roles ARIA)
- **CSS3** — Flexbox, CSS Grid, variables CSS, animations, media queries (responsive)
- **JavaScript Vanilla** — DOM, événements, IntersectionObserver, localStorage
- **Git & GitHub** — versioning, déploiement via GitHub Pages
- **Google Fonts** — Playfair Display (titres) & Inter (corps de texte)
- **Bootstrap Icons** — icônes (CDN)

## Fonctionnalités JavaScript implémentées

1. Dark Mode / Light Mode avec persistance via `localStorage`
2. Navbar dynamique (changement au scroll) + menu hamburger mobile
3. Animations fade-in au scroll via `IntersectionObserver`
4. Onglets du programme (Jour 1 / 2 / 3)
5. Filtrage dynamique des intervenants par thématique
6. Validation complète du formulaire d'inscription (regex email, téléphone, longueur du message)
7. Bouton retour en haut (apparition au scroll + smooth scroll)
8. Année dynamique dans le footer
9. Compteurs animés (chiffres clés) au scroll
